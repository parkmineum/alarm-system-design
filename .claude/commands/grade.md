`grader` 서브에이전트를 호출해서 현재 변경사항을 BE-C 평가 기준 100점 만점으로 채점해줘.

대상 범위: $ARGUMENTS (없으면 직전 커밋 ~ HEAD 또는 working tree 변경분 전체)

요구사항:
- 평가 6개 영역(요구사항 이해 20 / 설계·코드 25 / 안정성 20 / API DX 15 / 문서화 10 / Git 10) 각각 점수 + 근거 1~2줄
- 감점 사유는 파일·줄 단위로 인용
- "현재 만점 대비 부족한 것 Top 3" 액션 아이템으로 마무리

채점은 엄격하게. AI slop 발견하면 명시적으로 지적.
